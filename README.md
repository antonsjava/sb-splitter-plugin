
# sb-splitter-plugin

plugin ti disassemble spring boot jar to build layered docker image.


## Maven usage

```
	<plugin>
		<groupId>com.github.antonsjava</groupId>
		<artifactId>sb-splitter-plugin</artifactId>
		<version>1.0</version>
		<executions>
			<execution>
				<goals>
					<goal>split</goal>
				</goals>
			</execution>
		</executions>
	</plugin>
```
